# Quiz After Class

A hackathon-ready web application where faculty can generate quizzes after class and share them with students for instant assessment.

## Features
- Faculty quiz generation (subject, topic, difficulty)
- Auto-generated MCQs (AI + fallback)
- Shareable quiz link
- Student quiz attempt
- Instant scoring

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Styling: CSS
- Storage: In-memory (hackathon friendly)

## How to Run

### Backend
```bash
cd backend
npm install
node server.js
